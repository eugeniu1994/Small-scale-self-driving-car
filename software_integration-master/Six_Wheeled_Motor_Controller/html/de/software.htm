WILD THUMPER - Dokumentations & Software CD
Alle benötigten Software Tools für Ihren Wild Thumper finden Sie auf dieser Seite. Abhängig von den Sicherheitseinstellungen Ihres Browsers können Sie die Software direkt von dieser CD starten. Eventuell müssen Sie die Dateien zunächst in ein Verzeichnis auf Ihrer Festplatte speichern und diese dann von dort starten (klicken Sie dazu mit der rechten Maustaste auf den entsprechenden Link und selektieren Sie "Ziel Speichern unter" o.ä.).

HINWEIS: Sie müssen sich ggf. vorher als Administrator an Ihrem System anmelden um einige der Softwarepakete installieren zu können!


COMMON SOFTWARE

»Windows
 [TOOL] ADOBE READER 9

Sie benötigen den Adobe® Reader® um die Anleitung auf dieser CD ansehen zu können. Die neueste Version können Sie sich auch direkt von http://www.adobe.com/ downloaden.
Linux Anwender sollten besser den Distributionseigenen Paketmanager verwenden um den Adobe Reader zu installieren! Die Linux Version ist daher nicht auf dieser CD enthalten.

»Windows 32bit
»Windows 64bit
 Java 6 Runtime Environment

Sie benötigen das Java 6 Runtime Environment für die RobotLoader Software (siehe unten). Sie können je nachdem welches Betriebssystem Sie verwenden zwischen der normalen 32bit und der 64bit Version wählen.
Für Linux finden Sie im Software Verzeichnis selbstextrahierende oder RPM Versionen für 32 und 64bit Systeme. Unter Linux ist es allerdings ratsam, den Distributionseigenen Paketmanager zu benutzen um die aktuellste Java Version zu installieren! Bitte beachten Sie die Hinweise in der Wild Thumper Anleitung!

Neue Versionen des JRE erhalten Sie hier: http://java.sun.com/

Achtung: für Win98SE/ME oder Win2k ohne SP4 ist Java 6 leider nicht verfügbar! Kontaktieren Sie uns bitte sofern Sie noch ein solch altes Betriebssystem verwenden.

»RobotLoader

 RobotLoader v.2.5a

Dieses Programm wurde für den schnellen und einfachen Programmupload auf die Mikrocontroller des Wild Thumpers (und anderen Robotern wie z.B. dem RP6) entwickelt. Es ist in Java geschrieben und läuft direkt unter Windows und Linux. Sie müssen den RobotLoader nicht installieren - Sie können das Programm einfach in ein beliebiges Verzeichnis auf Ihrer Festplatte entpacken und von dort aus starten.
Wenn Sie Windows verwenden, installieren Sie bitte unbedingt den USB Treiber bevor Sie den RobotLoader starten!

Hinweis: Es wird das Java Runtime Environment (JRE) Version 6 oder neuer zur Ausführung benötigt!

»Speichern
 WT_Library und Beispielprogramme

Dieses Archiv enthält die Wild Thumper Funktionsbibliothek und die zugehörigen Beispielprogramme.

»Verzeichnis anzeigen
 Software Verzeichnis

Hier können Sie sich das komplette Software Verzeichnis anzeigen lassen. Auf der CD finden Sie auch die Version 3.6 von Mozilla Firefox und einiges andere.
WINDOWS SOFTWARE

»RACS

 WT APPLICATION Software

Software für Selbsttest und Drahtlose Steuerung, Robot-Application-Communication-Software.

»Installieren
 WinAVR Version 201001100

WinAVR ist eine Sammlung von Open Source Software Entwicklungstools für Atmel AVR Mikrocontroller. Es macht den AVR-GCC und andere Open Source Tools auch unter Windows benutzbar und ist sehr einfach zu installieren - folgen Sie einfach den Anweisungen des Installers.

Hinweis: Es sind mehrere Versionen von WinAVR im Software Verzeichnis auf der CD zu finden. Dies nur falls die neueste Version Probleme verursachen sollte.

»Win2k/XP/Vista/Win7 32 und 64bit
 USB Interface Treiber

Sie benötigen unter Windows diesen Treiber für das USB Interface (in modernen Linux Kerneln ist ein passender Treiber bereits enthalten). Die neueste CDM 2.12.06 Version ist für Win2k/XP/Vista/Win7 32 und 64 Bit verwendbar. ACHTUNG: Unter Windows Vista und Windows 7 ist bereits ein NICHT VOLLSTÄNDIGER standard Treiber vorhanden - dieser wird automatisch installiert wenn Sie das USB Interface das erste mal anschließen. Diesem Treiber fehlen allerdings einige Funktionen! Es ist daher ratsam den Treiber auf dieser CD unbedingt zu installieren BEVOR Sie das USB Interface an den PC anschließen! Wenn Sie dies nicht beachtet haben, müssen Sie über den Gerätemanager den standard Treiber durch den korrekten FTDI Treiber ersetzen.
Sie können den Treiber einfach mit dem CDM_Setup installieren BEVOR Sie das USB Interface das erste Mal anschließen! Wenn Sie den Treiber manuell installieren möchten, finden Sie die Treiberdateien in dem entsprechenden Verzeichnis. Die Installation wird in der Anleitung detailliert beschrieben und den Treibern sind auch Installationsanweisungen beigefügt (allerdings in Englisch).

Neuere Versionen der Treiber können auch direkt von FTDI heruntergeladen werden: http://www.ftdichip.com/
LINUX SOFTWARE

»Verzeichnis anzeigen
 avr-gcc, binutils, avr-libc und Patches

Dieses Verzeichnis enthält alles was Sie brauchen um den avr-gcc, binutils und die avr-libc zu kompilieren und zu installieren. Es ist auch ein automatisches Installationsskript beigefügt. Natürlich können Sie auch alles per Hand kompilieren und installieren wenn es mit dem Skript nicht klappen sollte. Dies wird sehr detailliert in der Anleitung beschrieben. Sie müssen sich auf jeden Fall den entsprechenden Abschnitt in der Anleitung durchlesen, denn das ist nicht ganz so einfach wie es mit WinAVR unter Windows ist. Man muss so einiges in die Kommandozeile eintippen! Wenn Sie hiermit Probleme haben sollten, schauen Sie bei arexx.com/forum vorbei.

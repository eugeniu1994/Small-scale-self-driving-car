Datenblatt-Verzeichnis

Die folgenden Dokumente richten sich hauptsächlich an fortgeschrittene Benutzer die die Funktionsweise der Hardware des Wild Thumper im Detail verstehen möchten. Aber auch für Einsteiger sind hier nützliche Informationen zu finden - z.B. Pinbelegungen und Erläuterung der Prozessor Register.

Hier können Sie sich das Datenblatt-Verzeichnis ansehen.

Es enthält die folgenden Dateien:
  • ATMEGA644PA.pdf - Atmel ATMEGA64 8-Bit RISC Microcontroller
  • FT232R.pdf - FTDI FT232RL USB UART

  • JSR6WD Motor specs.pdf - JSR6WD Motor spezifikationen
  • 4094D.pdf - 4094 Shiftregister
  • APC220.pdf - Drahtlose HF-Transceiver

  • CD40107.pdf - NAND Buffer/Driver
  • L9440.pdf - Voltage regulator very low drop
  • LM35.pdf - Temperature sensor

  • LM317.pdf - Voltage regulator
  • MCP1541.pdf - Voltage reference chip
  • TLC27M2CD.pdf - DUAL OPAMP

  • TLC5206-2.pdf - H-Bridge DC motors
  • TSOP34836.pdf - IR receiver UART
  • WT_CHASSIS.pdf - Wild Thumper Chassis

  • WT_Motor_2.pdf - Mehr WT motor spezifikations
  • SS411P.pdf - HAL sensor
